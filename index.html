My Projects

Movie Recommendation System

My movie recommendation system is built using data from the IMDb Dataset of Top 1000 Movies and TV Shows, sourced from Kaggle and provided by Harshit Shankhdhar. This system helps users find the perfect movie by analyzing release year, CBFC rating, genre, publication year, and IMDb rating.

Number Recognition System

The Number Recognition System leverages a convolutional neural network (CNN) trained on the MNIST dataset to accurately identify handwritten digits (0–9). Users can interact with the system through a simple web interface that allows them to either draw or upload digit images for predictions.

Firework-vs-Angos

This is an exciting shooting game where players use their mouse to launch fireworks at angos. As your score increases, so does the difficulty—the number and HP of angos rise. Fortunately, your weapon evolves too, becoming more powerful and visually impressive. My personal record is 1205 points—can you beat it?

Online Study Room

The Online Study Room creates a calm and focused environment to help you study effectively. It features a serene background paired with a playlist of 15 carefully selected Japanese instrumental tracks to set the perfect mood. Additionally, a minimalist white timer is included to manage productive periods and breaks.
